プライバシーポリシー

最終更新日：2025年12月

1. はじめに

AruCall（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用するかについて説明しています。

2. 収集する情報

本アプリでは、以下の情報を収集・保存します。

2.1 認証情報

  • 匿名ユーザーID：Supabaseによる匿名認証により生成される一意の識別子
  • 本アプリは匿名ログインを使用しており、メールアドレス、パスワード、その他の個人を特定できる情報は収集しません。

2.2 アプリ使用データ

  • 飲酒杯数：ユーザーが記録した累計の飲酒杯数
  • 表示名：アプリ内で使用するニックネーム（デフォルトは「ビールくん」）
  • キャラクターレベル：飲酒杯数に基づいて算出されるレベル情報
  • アプリ設定：通知設定などのアプリ内設定（デバイスのローカルストレージに保存）

2.3 デバイス情報

  • 通知権限：iOSデバイスでの通知機能を使用するために必要な権限
  • プッシュ通知トークン：Firebase Cloud Messagingによるプッシュ通知を送信するために必要なデバイストークン
  • トラッキング許可状態（iOS）：App Tracking Transparencyによる広告トラッキング許可状態（ユーザーが許可した場合のみ）

2.4 広告関連の情報

本アプリでは、Google AdMobを使用して広告を表示しています。AdMobは以下の情報を収集する場合があります。

  • 広告識別子：デバイスの広告ID（ユーザーがトラッキングを許可した場合）
  • 広告の表示・クリック情報：広告の効果測定のための統計情報
  • デバイス情報：デバイスの種類、OSバージョン、言語設定など（広告の配信に使用）

これらの情報は、Google AdMobのプライバシーポリシーに従って処理されます。

2.5 収集しない情報

本アプリでは、以下の情報は収集していません。

  • 位置情報
  • 連絡先情報
  • 写真・画像
  • カメラ・マイクのアクセス

3. 情報の保存と管理

3.1 データ保存場所

本アプリで収集した情報は、以下の場所に保存されます。

  • Supabaseデータベース：飲酒杯数、表示名、キャラクターレベルなどのアプリデータ（https://supabase.com）
  • デバイスローカルストレージ：通知設定などのアプリ設定（SharedPreferences）
  • Firebase：プッシュ通知トークン（Google LLCが管理）
  • Google AdMob：広告関連の統計情報（Google LLCが管理）

データは暗号化され、安全に管理されます。

3.2 データアクセス制御

  • SupabaseデータベースにはRow Level Security (RLS)が適用されており、各ユーザーは自分のデータにのみアクセスできます。他のユーザーのデータを閲覧することはできません。
  • デバイスローカルストレージのデータは、そのデバイス内でのみアクセス可能です。

4. 情報の使用目的

収集した情報は、以下の目的でのみ使用されます。

  1. アプリ機能の提供：飲酒杯数の記録と管理
  2. キャラクター成長システム：飲酒杯数に基づくレベル計算と表示
  3. データの永続化：アプリ再起動後もデータを保持するため
  4. プッシュ通知機能：お知らせを送信するため
  5. 広告の表示：アプリの無料提供を維持するための広告配信

5. 第三者サービス

本アプリは以下の第三者サービスを使用しています。

5.1 Supabase

  • 提供元：Supabase Inc.
  • 用途：認証、データベース保存
  • プライバシーポリシー：https://supabase.com/privacy

5.2 Firebase（Google）

  • 提供元：Google LLC
  • 用途：プッシュ通知の送信（Firebase Cloud Messaging）
  • 収集する情報：プッシュ通知トークン、デバイス情報
  • プライバシーポリシー：https://policies.google.com/privacy

5.3 Google AdMob

  • 提供元：Google LLC
  • 用途：広告の表示と配信
  • 収集する情報：広告識別子、デバイス情報、広告の表示・クリック情報（ユーザーがトラッキングを許可した場合）
  • プライバシーポリシー：https://policies.google.com/privacy

5.4 Google Fonts

  • 提供元：Google LLC
  • 用途：アプリ内で使用するフォントの提供
  • 収集する情報：フォントのリクエスト時にIPアドレスが記録される場合があります
  • プライバシーポリシー：https://policies.google.com/privacy

6. データの共有

本アプリは、以下の場合を除き、収集した情報を第三者と共有することはありません。

  • 法的な要求がある場合
  • ユーザーの同意がある場合

7. セキュリティ

本アプリは、データの安全性を確保するために以下の対策を講じています。

  • Supabaseの暗号化通信（HTTPS）
  • Row Level Security (RLS)によるデータアクセス制御
  • 匿名認証による個人情報の最小化
  • Firebaseのセキュアな通信プロトコル
  • デバイスローカルストレージへの設定情報の安全な保存

ただし、インターネット経由でのデータ送信は100%安全であることを保証することはできません。ユーザーは、本アプリの使用により生じるリスクを認識し、自己責任において使用するものとします。

7.1 広告トラッキングについて

本アプリでは、App Tracking Transparency（ATT）を使用して、広告トラッキングの許可を求めます。ユーザーがトラッキングを許可しない場合でも、アプリの基本機能は利用可能です。ただし、広告のパーソナライゼーション機能は制限される場合があります。

広告トラッキングの設定は、iOSの設定アプリからいつでも変更できます。

8. 子どものプライバシー

本アプリは、飲酒を記録するアプリの性質上、20歳未満の方を対象にしていません。そのため、意図的に13歳未満の子どもから個人情報を収集することはありません。

9. プライバシーポリシーの変更

本プライバシーポリシーは、予告なく変更される場合があります。重要な変更があった場合は、アプリ内通知またはその他の方法でお知らせします。変更後もアプリを継続して使用される場合、変更後のプライバシーポリシーに同意したものとみなされます。
